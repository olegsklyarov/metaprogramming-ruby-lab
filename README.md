# Metaprogramming Ruby Lab

Практические эксперименты по книге *Metaprogramming Ruby (Paolo Perrotta)*.

Цель:
- пройти книгу до конца;
- запускать все примеры руками;
- исследовать интересные механизмы Ruby глубже;
- сохранять собственные эксперименты.
