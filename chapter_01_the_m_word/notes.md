# Chapter 1

## Что узнал:
- Ruby в runtime полностью сохраняет всю информацию о переменных, объектах, классах и тп.
- В Ruby можно делать интроспекцию (introspection) - получать информацю о классах, объектах, методах и переменных.
- attr_reader - макрос Ruby, который автоматически создает метод для чтения значения переменной экземпляра указанной в аргументе
- Assignment method syntax — специальный синтаксис вызова методов, имя которых заканчивается на =.
